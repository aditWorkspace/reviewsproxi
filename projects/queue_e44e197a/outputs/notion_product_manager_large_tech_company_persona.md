## Trigger Map

| Trigger Type | Specific Conditions |
|---|---|
| **Converts when** | Product demonstrates clear reliability metrics and uptime guarantees / Support team responds quickly with helpful, specific guidance / UI is intuitive enough to complete core tasks without extensive training |
| **Abandons when** | Setup requires technical knowledge beyond PM expertise / Interface has confusing navigation or unclear error messages / Customer support is slow or provides generic responses |
| **Trusts when** | Transparent about system limitations and known issues / Provides detailed documentation without requiring technical background / Shows evidence of responsive customer support in reviews |
| **Suspicious when** | Pricing model has hidden fees or removes features after payment / Product requires extensive technical setup not mentioned upfront / Support documentation is sparse or overly technical |

### Deal Breakers (Immediate Exit)

- 🚫 App crashes frequently or causes data loss
- 🚫 Customer support is unresponsive or ineffective
- 🚫 Poor user interface with unclear error messages
- 🚫 Requires technical knowledge beyond PM skillset
- 🚫 Predatory subscription models that remove paid content
