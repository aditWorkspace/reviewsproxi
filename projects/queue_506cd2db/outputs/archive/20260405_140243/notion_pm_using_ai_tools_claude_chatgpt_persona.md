## Trigger Map

| Trigger Type | Specific Conditions |
|---|---|
| **Converts when** | Trusted colleague provides specific recommendation with reliability evidence / Tool demonstrates ability to handle large datasets without performance issues / Clear documentation shows exactly how to validate and debug automated outputs |
| **Abandons when** | Discovers tools require extensive manual validation due to unreliable automated processing / Finds evidence of feature removal in upgrades without clear business justification / Encounters complex workflows that are easily disrupted by small configuration errors |
| **Trusts when** | Vendor provides transparent data security and governance policies / Tool outputs include clear traceability for audit and debugging purposes / Documentation includes specific examples of error handling and recovery procedures |
| **Suspicious when** | Marketing promises sound too good to be true without technical details / Pricing information requires signup before showing actual costs and limitations / User reviews mention silent failures or data corruption issues |

### Deal Breakers (Immediate Exit)

- 🚫 Software that fails silently or produces empty outputs without clear error messages
- 🚫 Tools that require bouncing between multiple applications for basic ETL and visualization tasks
- 🚫 AI systems that lack traceability making it impossible to audit and debug errors
- 🚫 Subscription models that remove core features in upgrades without transparent justification
- 🚫 Interfaces that require extensive learning curves without comprehensive tutorial support
